# Testausserveri.fi

This repository holds the website of Testausserveri ry and DNS configurations of testausserveri.fi domain. 

Keep in mind that we are planning a redesign of our website during autumn-winter 2021, therefore we don't recommend you making major changes just yet.

DNS configurations are stored in the [`dns/dnsconfig.js`](https://github.com/Testausserveri/testausserveri.fi/blob/main/dns/dnsconfig.js) file and are directly synchronized to Cloudflare. For more advanced configurations please seek information from [DNSControl documentation](https://stackexchange.github.io/dnscontrol/js).
